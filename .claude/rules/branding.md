# Branding\n\n- **Three source assets** (all at project root, 1024x1024 RGB):\n  - `praxis-favicon.png` - cursor+spark silhouette only → toolbar/favicon (16px)\n  - `praxis-icon.png` - dark rounded-rect with capture frame + cursor → manifest icons (32/48/128px)\n  - `praxis-logo.png` - dark circle with capture frame + cursor → UI header badges (32px+)\n- All source PNGs are RGB without alpha — Pillow converts near-white (>=240) to transparent during resize\n- Extension icons: `public/icons/icon-{16,32,48,128}.png` + `logo-32.png`\n- Regenerate icons: `node scripts/create-placeholder-icons.cjs` (requires macOS `sips`)\n- Header badges use `chrome.runtime.getURL('icons/logo-32.png')` in popup and editor App.tsx\n- Favicons added to both editor and popup index.html pages\n- Key lesson: don't try to use one logo for all sizes — favicon/icon/logo each need different detail levels
